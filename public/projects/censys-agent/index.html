Censys Summarization Agent – Project Docs

ReactFastAPIPythonDocker

Overview

An AI-powered, full‑stack application that ingests Censys host data and turns it into analyst‑grade summaries: risks, insights, and next actions. Ships with a modern React UI, a FastAPI backend, optional LLM rewrite, metrics, and handy export tools.

End‑to‑End Flow

Features

  • Import JSON/CSV from Censys; batch uploads supported
  • Risk scoring, service fingerprinting, and flagged exposures
  • LLM rewrite for executive‑friendly summaries (optional)
  • Exports: Markdown, PDF, CSV of findings
  • Metrics: run stats, time-to-summary, flagged count
  • API-first with Swagger/OpenAPI

Quick Start (Windows PowerShell)

# 1) Backend
python -m venv .venv; . .venv/Scripts/Activate.ps1
pip install -U pip wheel fastapi uvicorn[standard] pydantic
$env:PORT=8000; uvicorn app.main:app --host 0.0.0.0 --port $env:PORT

# 2) Frontend
npm install; npm run dev

# Open http://localhost:5173

Docker (Optional)

docker build -t censys-agent .
docker run --rm -p 8000:8000 censys-agent

API

  • POST /api/ingest – upload Censys data
  • GET /api/summary – get computed summaries
  • POST /api/rewrite – optional LLM rewrite
  • GET /api/metrics – run and performance metrics

Testing

# API smoke
curl -s http://localhost:8000/health

# Frontend lint/build
npm run lint; npm run build
